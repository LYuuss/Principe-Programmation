# Service Web - Implementation d'une première api SOAP, en java  
Mise en oeuvre d'une simple application impliquant un serveur répondant à des requêtes relativement basique (e.g. convertir de l'euro en bitcoin et inversement).  

# Outils utilisé :  
- IntelliJ
- Java 1.8  
- packages :  
    - javax.xml.ws.Endpoint;  
    - javax.xml.bind.annotation.XmlRootElement;  
    - java.io.Serializable;  
    - javax.jws.WebService;
